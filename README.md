## Steps to run locally

 
1. Clone the project [covid-spa](https://github.com/lizandro94/covid-spa)
```bash

git clone https://github.com/lizandro94/covid-spa

```
  2. Open repo folder
```bash

cd covid-spa/

```

3. Install dependencies
```bash

npm install

```
5. Run the development server
```bash

npm start

```

To run tests:
```bash

npm test

```

 Press `a` to run all the tests.
